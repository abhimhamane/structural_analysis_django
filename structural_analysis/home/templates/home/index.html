How to use this Website?

Features Available:

  • Beam Analysis
    • Shear Force Diagram
    • Bending Moment Diagram
    • Deflection Diagram
  • What If Analysis
    • Support Effect
    • Loading Effect
    • Compare

options for other structural elements like Truss, Frames, Columns will be added in newer versions.

Beam Analysis

Beam Analysis- helps analyse beams using two methods:
  1. Stiffness Matrix Method
  2. Slope Deflection Method

One can get the outpit of analysis in form of SFD, BMD, Deflection Diagram and Detailed analytical solutions. Other Analysis methods like Moment Distribution Method and Kanni's Method will be made available in later versions.

What If Analysis

What If Analysis - Helps develop understanding by tweaking something and comparing SFD and BMD.
  1. Support Effect - What happens when support conditions are changed?
  2. Loading Effect - What happens when loading conditions are changed?
  3. Compare Mode - Compare two different beams (varying support and loading condition)

One can get the outpit of analysis in form of SFD, BMD, Deflection Diagram and Detailed analytical solutions. Other Analysis methods like Moment Distribution Method and Kanni's Method will be made available in later versions.

Sympy Beam Analysis Workflow

Sympy Workflow is implemented under the hood. It can be seen at Sympy Beam Analysis Workflow Jupyter Notebook.
